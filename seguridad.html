Windows Defender Antivirus

Anteriormente conocido como Microsoft Security Essentials, Windows Defender es un antivirus integrado que protege contra software malicioso y spyware en correos electrónicos, aplicaciones y la web.

Las características de Microsoft Defender son: protección contra virus y amenazas, protección de cuentas de Microsoft que están en el equipo, cortafuegos y protección de red, control de aplicaciones y explorador y seguridad de hardware del dispositivo.

Windows Defender se basa en GIANT AntiSpyware, que fue desarrollado originalmente por GIANT Company Software. Microsoft adquirió GIANT el 16 de diciembre de 2004. Aunque el original AntiSpyware GIANT era compatible con versiones anteriores de Windows, el soporte para la línea de Windows 9x de sistemas operativos fue abandonado más tarde. Sin embargo, Sunbelt Software, que originalmente era socio de GIANT, vende un producto basado en la misma tecnología denominada CounterSpy que todavía tiene soporte para anteriores sistemas operativos de Microsoft.

Originalmente tenía el nombre de Windows Defender, pero se lanzó una versión para Android y el nombre cambió a Microsoft Defender. Actualmente la versión de Microsoft Defender trae muchas mejoras, inclusive tiene integración con IA para el análisis de Virus en la Nube, está disponible en Windows 10 por defecto, Windows 8 y 8.1, Windows 11, Windows Server 2016 y Windows Server 2019.

Windows Hello

Proporciona un método seguro para iniciar sesión en dispositivos Windows mediante reconocimiento facial, huella digital o un PIN. Utiliza tecnología biométrica para hacer el acceso más seguro y personal.

Windows Hello puede implementar diferentes métodos de reconocimiento dependiendo del hardware que tengas en tu ordenador. Si tienes una cámara con infrarrojos, podrás configurar su sistema de reconocimiento facial. Y si tienes un teclado con sensor de huella dactilar, podrás utilizar esta huella para iniciar sesión.

Control de Cuentas de Usuario (UAC)

UAC ayuda a prevenir cambios no autorizados en el sistema operativo solicitando permiso o una contraseña de administrador antes de permitir acciones que podrían afectar la operación del sistema o cambiar configuraciones que requieren privilegios elevados.

El control de cuentas de usuario de Windows tiene cuatro niveles de seguridad, a cada cual de ellos más restrictivo. Su funcionamiento es sencillo, sólo tienes que entrar en la configuración de este parámetro y mover una barra seleccionadora al nivel de seguridad que quieras.

Notificarme siempre: Te notificará cuando los programas que tienes en Windows intenten instalar otras aplicaciones o realizar cambios en el equipo. También se te notificará cuando realices cambios en la configuración, y se inmovilizarán otras tareas hasta que respondas a la ventana de autorización.

Notificarme solamente cuando una aplicación intente realizar cambios en el equipo: Es el nivel de seguridad que viene configurado por defecto. Sólo te notificará cuando los programas que tienes en Windows intenten instalar otras aplicaciones o realizar cambios en el equipo. NO te avisará cuando tú realices cambios en la configuración, pero sí inmovilizará otras tareas hasta que respondas en la ventana emergente.

Notificarme solo cuando una aplicación intente realizar cambios en el equipo (no atenuar el escritorio): Básicamente como el nivel superior, avisándote cuando los programas que tienes en Windows intenten instalar otras aplicaciones o realizar cambios en el equipo, pero no notificándote cuando seas tú quien realice cambios en la configuración. La gran diferencia es que aquí ya no se inmovilizará el resto de tareas, podrás seguir trabajando con el aviso de fondo.

No notificarme nunca: Es la configuración menos recomendada, porque desactiva el UAC. Esto quiere decir que no te notificará cuando las aplicaciones intenten instalar otro software o realizar cambios en el equipo, ni cuando seas tú quien realice los cambios. Tampoco atenuará la ventana en ningún momento impidiendo que puedas seguir trabajando.

Windows Defender Firewall

Ayuda a proteger las computadoras de acceso no autorizado, bloqueando el tráfico de red potencialmente peligroso o sospechoso.

Windows Defender Firewall es una parte integral del sistema de seguridad de Microsoft Windows, diseñada para proteger las computadoras de accesos no autorizados mientras permiten conexiones de red seguras. Es el sucesor del Firewall de Internet Connection Firewall (ICF) y Windows Firewall, y ofrece capacidades ampliadas y una integración más profunda con el sistema operativo Windows.

La importancia de Windows Defender Firewall en la seguridad de un sistema Windows no puede subestimarse. Actúa como una barrera entre la computadora y potenciales amenazas externas, evitando el acceso no autorizado mientras permite una comunicación segura y necesaria. Su capacidad para ser configurado de manera detallada permite a los usuarios y administradores de sistemas afinar la seguridad para adaptarse a las necesidades específicas de su entorno operativo.

Actualizaciones de seguridad regulares

Microsoft libera constantemente actualizaciones y parches para sus sistemas operativos, lo que incluye mejoras de seguridad que protegen contra vulnerabilidades recién descubiertas.

Las actualizaciones de seguridad regulares en Windows son una parte fundamental del mantenimiento de la seguridad y la estabilidad del sistema operativo. Microsoft las utiliza para proporcionar parches para vulnerabilidades, mejoras en la funcionalidad del sistema y actualizaciones de características que ayudan a proteger los computadores contra amenazas recientes y emergentes.

Las actualizaciones de seguridad son correcciones y mejoras que Microsoft desarrolla y distribuye para abordar problemas específicos identificados en diferentes versiones de Windows y otros productos de software de Microsoft. Estas actualizaciones son críticas para cerrar brechas de seguridad que podrían ser explotadas por malware, ransomware, y ataques cibernéticos.

Secure Boot

Asegura que solo software confiable con certificación de Microsoft pueda cargarse durante el proceso de inicio, lo que ayuda a prevenir rootkits y otros tipos de malware que infectan el arranque del sistema.

Secure Boot es una característica de seguridad que forma parte de la especificación UEFI (Unified Extensible Firmware Interface), un estándar moderno de firmware para computadoras que ha reemplazado al antiguo BIOS (Basic Input/Output System). Su función principal es asegurar que solo el software autorizado por el fabricante del dispositivo, especialmente el sistema operativo, se cargue durante el proceso de inicio de la computadora. Esto ayuda a proteger el sistema contra ciertos tipos de malware, como los rootkits, que pueden infectar el proceso de arranque antes de que el sistema operativo y el antivirus tengan la oportunidad de cargar y tomar control.

Cuando una computadora con Secure Boot se enciende, el firmware verifica la firma digital de cada pieza de software de arranque, incluyendo el cargador de arranque del sistema operativo y otros componentes de firmware. Esta verificación se realiza para asegurar que no han sido alterados o reemplazados por software malicioso. Si las firmas son válidas, el sistema se inicia normalmente; si alguna verificación falla, el arranque se detiene, y el sistema muestra un mensaje de error, lo que previene la ejecución de software potencialmente dañino.
